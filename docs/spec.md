## Version 1.0

Hermes-based Developer Knowledge OS for NextPlatform Ecosystem.

### Core Modules

- Project Manager
- Repository Manager (GitHub sync)
- Document Manager
- Memory Manager (5 types)
- Skill Manager
- Knowledge Graph
- Search Engine (keyword/semantic/hybrid/RAG)
- Reflection Engine
- Hermes Integration (MCP + context sync)
